# Changelog

## [0.1.0] — 2026-05-13

### Added

- تطبيق `haulage_mgmt` لموديول **Haulage Logistics** على ERPNext.
- أنواع مستندات: شاحنات، سائقون، خطوط سير، أنواع مصروفات، طلبات شحن، تجهيز شحنات، رحلات مع جداول فرعية (شحنات، أحداث، مصروفات).
- ربط العملاء عبر `Customer`، واختصارات فواتير البيع من طلب الشحن والرحلة.
- تقارير: ملخص مالي للرحلات، أداء السائقين، أداء الشاحنات.
- نسق طباعة **Haulage Trip Dispatch** وربطه كافتراضي لرحلة الشحن.
- إعدادات لوجستية: صنف فاتورة افتراضي، حساب دائن لقيد مصروفات الرحلة.
- قيد يومية (مسودة) من مصروفات الرحلة، وربطه بالرحلة.
- Workspace مكتب، وتوسيع لوحة **Customer** لطلبات الشحن.
- مزامنة حالات التجهيز/الطلب وحالة الشاحنة (مشغولة/متاحة) مع الرحلة.
- مهمة مجدولة يومية لتذكير وثائق الأسطول (ToDo).
- دور **Fleet Manager** وصلاحياته على أنواع المستندات.
